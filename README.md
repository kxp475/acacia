# :deciduous_tree: Acacia

**Acacia is a lifestyle app designed to organize your daily life.**

- :pencil2: **Customizable**: Acacia allows you to selected activities that apply to your daily routine.

- :bar_chart: **Smart**: It keeps track of your activities every day, allowing you to see your progress.

- :gem: **Beautiful**: Built on the Flutter framework, Acacia has an modern and intuitive UI.

# :iphone: Installation
## Prerequisites
- An Android (ARM processor) or iOS (iPhone 4S or newer) device.
- Must be running either:
  - **Android** Jelly Bean, v16, 4.1.x or newer
  - **iOS** 8 or newer

# :computer: For Developers
To contribute to the Acacia app you must have a viable Flutter programming environment.

## Flutter Environment
- [Flutter SDK](https://flutter.dev/docs/get-started/install) version newer than 2.1.0 is required.
- A IDE environment like [VS Code](https://code.visualstudio.com) or [Android Studio](https://developer.android.com/studio).
- Emulation environment.
  - Android emulation can be achieved with [Android Studio](https://developer.android.com/studio) or [Android Command Line Tools](https://developer.android.com/studio#command-tools).
  - iOS emulation requires [Xcode](https://developer.apple.com/xcode/).

## Getting the Source
Once, the environment is set up you can clone the repo with:
```bash
git clone https://github.com/kxp475/acacia.git
cd acacia
```
Before you can run the Acacia app on your emulator, you must download the requirements as specified in the [pubspec.yaml](pubspec.yaml). To do this, just run:
```bash
flutter pub get
```

Once you have set up an emulator for the system of your choice and you got the required plugins, you can run the following command to view the Acacia app on the emulator:
```bash
flutter run
```

The Acacia app utilizes the [Flutter SDK](https://flutter.dev) for all aspects of development, from front-end to back-end. The database is implemented using Google's [Firebase](https://firebase.google.com/).
